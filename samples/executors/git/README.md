# Git Executor

GitExecutorを使ったリポジトリ操作サンプル。

## Files

| File | Description |
|------|-------------|
| `basic.json` | 基本Git操作（clone, status, list_branches, log） |
| `all_operations.json` | Git全操作網羅 |

## Supported Operations

| Operation | Description |
|-----------|-------------|
| `init` | リポジトリ初期化 |
| `clone` | リポジトリクローン |
| `open` | 既存リポジトリを開く |
| `status` | ステータス確認 |
| `diff` | 差分表示 |
| `log` | コミット履歴 |
| `branch` | ブランチ操作 |
| `checkout` | ブランチ切り替え |
| `fetch` | リモート取得 |
| `commit` | コミット作成 |
| `push` | リモートへプッシュ |
